What Is the Attribution Option?

The showAttribution option adds a small, unobtrusive link at the very bottom of the widget footer ‚Äî below the disclaimer and reset button. It is off by default and must be explicitly enabled. Enabling it is a simple way to credit the open-source project that powers your widget.

Opt-in only Attribution is never shown unless you pass showAttribution: true. There is no tracking, no external requests, and no default branding.

Code Used on This Page

The widget on this page is initialised with all three attribution options. Copy and adapt the snippet below for your own site. 

// Minimal ‚Äî just turn it on (uses defaults)
AccessibilityWidget.init({
  showAttribution: true
});

// Full ‚Äî custom URL and label text
AccessibilityWidget.init({
  showAttribution:  true,
  attributionUrl:   'https://www.advertease-tech.com/',
  attributionText: 'Built with Free Accessibility Menu‚Ñ¢'
});
