Accessibility Widget – Initial Language Demo

This page demonstrates initialising the accessibility widget with defaultLanguage: 'he'. The floating button opens the panel in Hebrew from the very first render.

About This Demo

Pass any supported two-letter BCP-47 language code to defaultLanguage when calling AccessibilityWidget.init(). The widget will render its menu title, all feature labels, and the language selector in that language immediately — no user interaction required.

Ten languages are built in: English, Hebrew, Chinese, Spanish, Arabic, Portuguese, French, German, Japanese, and Russian. Register additional languages at runtime with registerLanguage().

Usage

Initialise the widget in Hebrew by passing defaultLanguage: 'he':

AccessibilityWidget.init({
  defaultLanguage: 'he',
});
