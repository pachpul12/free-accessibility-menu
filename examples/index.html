Widget Features

The accessibility widget includes the following features:

  • High Contrast -- increases contrast between text and backgrounds
  • Dark Mode -- switches to a dark colour scheme to reduce eye strain
  • Font Size -- adjustable text scaling from default up to 175%
  • Dyslexia-Friendly Font -- applies a typeface designed for easier reading
  • Underline Links -- makes hyperlinks more visually distinct
  • Hide Images -- removes images for distraction-free reading
  • Focus Outline -- highlights the currently focused interactive element
  • Text Spacing -- increases line height, letter and word spacing for readability
  • Pause Animations -- stops CSS animations and transitions across the page
  • Large Cursor -- enlarges the mouse cursor for better visibility
  • Highlight Headings -- visually emphasizes headings for easier navigation
  • Invert Colors -- inverts the page colours using CSS filters
  • Reading Guide -- displays a horizontal bar following your cursor to aid reading
  • Text to Speech -- click any text to have it read aloud using the Web Speech API

Multilingual Support

The widget supports multiple languages out of the box. In this demo you can switch between English and Hebrew, which also demonstrates the right-to-left layout support. Additional languages can be registered at runtime through the public API.

Persistent Settings

Your preferences are stored in the browser's localStorage, so they persist across page reloads. Use the "Reset All" button inside the widget to clear saved settings and return to the defaults.
